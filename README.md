# Paws-N-Purpose
Paws 'n Purpose is a web application which serves as a fundraising platform that is specialized in giving a proper channel for animals to receive help or care through funds. It introduces features that are tailored to helping animals in need such as donation drives for medical care and basic needs, sponsorship options.

## Setup

### Frontend (React)

i. Nagivate inside the react app

    cd paws-n-purpose-frontend
    
ii. Install dependencies 

    npm install
    npm install @mui/material @emotion/react @emotion/styled @mui/icons-material

    
### Backend (Springboot)

i. Navigate inside the springboot project
    
    cd paws-n-purpose-backend\paws-n-purpose-backend

ii. Install dependencies

    mvnw.cmd clean install
